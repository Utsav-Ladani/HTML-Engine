HTML Engine

This is the simple demo of the HTML Engine. 

    => Basic idea
    The HTML engine to combine mutiple html file and produce a single html file.

    => How to configure project?
    make something cli node run-engine file name
    libs like ejs etc

    => Strcture of input
    files with ext .html in folder views

    => output
    single file with .html ext

    process done through nodeJS

    syntax for embed html: { name.html }

    => version v1.0.0
    take input files combine it and produce single file
    no loops, no variables, just basic
